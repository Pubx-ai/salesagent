Curation Services Configuration

Connect to your external Catalog, Sales, and Activation services. These services manage audience segments, deal records, and SSP activations respectively.

External Services

The curation adapter delegates all data persistence to external services. No data is stored in the Prebid Sales Agent database for this adapter.

Service URLs

Base URL of the Curation Catalog service that provides audience segments.
Base URL of the Curation Sales service that manages sale/deal records.
Base URL of the Curation Activation service that triggers SSP deals.

Pricing Configuration

Multiplier applied to base CPM.
Minimum floor CPM for all segments.
Cap on the suggested CPM.

Advanced

Request timeout for calls to curation services.
